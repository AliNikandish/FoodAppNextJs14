## Installation

#### Clone the project

```bash
  git clone https://github.com/AliNikandish/FoodAppNextJs14.git
```

#### Go to the project directory

```bash
  cd FoodAppNextJs14
```

#### Install FoodApp with npm

```bash
  npm install
```

#### env:
```bash
  add variable to env file
```
#### Run:
```bash
  npm run dev
```
